# 힌트 및 풍선 도움말 기능


## *힌트 기능*

열의 너비가 부족하여 셀안의 모든 내용이 보여지지 않을 때, 마우스커서가 올라간 셀에 한하여 셀의 크기를 임시적으로 늘려 내용을 보여주는 기능을 제공합니다.

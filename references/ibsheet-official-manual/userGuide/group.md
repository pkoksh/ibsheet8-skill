# 그룹핑 기능

그룹핑 기능이 활성화 된 시트에서 사용자가 드래그를 통해 조회된 데이터를 그룹지어 볼 수 있습니다.

그룹핑 기능은 그룹핑 기능이 추가된 화면에서만 사용 가능합니다.


## *그룹핑*

특정 열을 기준으로 그룹핑된 데이터를 확인할 수 있습니다.

![그룹기능](/assets/imgs/basic_group.gif "그룹핑 기능")

## *그룹핑된 데이터에서 소계 계산*

그룹된 데이터 내에서 소계 등의 계산 결과를 확인할 수 있습니다.

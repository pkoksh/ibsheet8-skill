# hideFilterDialog ***(method)***
> Hides the header filter dialog icon.


### Syntax
```javascript
boolean hideFilterDialog();
```

### Return Value
***none***

### Example
```javascript
// Hides the header row filter dialog icon. 
sheet.hideFilterDialog();
```

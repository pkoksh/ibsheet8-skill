# version ***(method)***
> 인스턴스화된 시트 객체의 버전을 확인합니다.

### Syntax
```javascript
string version();
```

### Return Value
***string***: 인스턴스화된 시트 객체의 버전 및 로드된 플러그인들의 버전 문자열

### Example
```javascript
// 인스턴스화된 시트 객체의 버전 확인
sheet.version();
```

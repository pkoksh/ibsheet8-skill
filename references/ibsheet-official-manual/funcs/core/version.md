# version ***(method)***
> Checks the version of the instantiated sheet object.

### Syntax
```javascript
string version();
```

### Return Value
***string***: Version of the instantiated sheet object and loaded plugins as a string

### Example
```javascript
// Check the version of the instantiated sheet object
sheet.version();
```

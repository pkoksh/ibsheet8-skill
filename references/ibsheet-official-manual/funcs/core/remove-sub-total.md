# removeSubTotal ***(method)***
> Removes all subtotal/cumulative total rows existing in the sheet.

### Syntax
```javascript
void removeSubTotal();
```

### Return Value
***none***

### Example
```javascript
// Remove subtotal/cumulative total rows
sheet.removeSubTotal();
```

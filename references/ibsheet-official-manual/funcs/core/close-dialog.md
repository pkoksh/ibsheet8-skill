# closeDialog ***(method)***
> Closes the dialog or `Menu` provided by the sheet.

### Syntax
```javascript
void closeDialog();
```

### Return Value
***none***

### Example
```javascript
//Closes the open menu/dialog.
sheet.closeDialog();
```

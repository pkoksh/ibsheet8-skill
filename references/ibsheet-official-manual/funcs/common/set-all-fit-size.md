# setAllFitSize ***(method)***

> Uses fitSize to adjust the width of all columns. 


### Syntax
```javascript
void setAllFitSize();
```

### Return Value
***none***

### Example
```javascript
// Uses fitSize to adjust the width of all columns.
sheet.setAllFitSize();
```

# getSheetHeight ***(method)***

> Returns the height of the sheet. 


### Syntax
```javascript
void getSheetHeight();
```

### Return Value
***number*** Height of the sheet


### Example
```javascript
// Returns the height of the sheet.
sheet.getSheetHeight();
```

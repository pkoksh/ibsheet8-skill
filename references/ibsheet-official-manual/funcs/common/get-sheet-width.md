# getSheetWidth ***(method)***

> Returns the width of the sheet. 


### Syntax
```javascript
void getSheetWidth();
```

### Return Value
***number*** Width of the sheet

### Example
```javascript
// Returns the width of the sheet.
sheet.getSheetWidth();
```

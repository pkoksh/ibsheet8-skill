# getSheetFocused ***(method)***

> Returns the currently focused sheet object. 


### Syntax
```javascript
void getSheetFocused();
```

### Return Value
***Sheet*** Sheet object


### Example
```javascript
// Returns the currently focused sheet object.
sheet.getSheetFocused();
```

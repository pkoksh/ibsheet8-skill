# getFooterRows ***(method)***

> 푸터 영역에 있는 푸터 데이터 로우 객체들의 배열을 반환합니다. 


### Syntax
```javascript
void getFooterRows();
```

### Return Value
***array[row object]*** : 데이터 로우 객체 배열

### Example
```javascript
// 푸터 영역에 있는 푸터 데이터 로우 객체들의 배열을 반환합니다.
sheet.getFooterRows();
```

# getFooterRows ***(method)***

> Returns an array of footer data row objects in the footer area. 


### Syntax
```javascript
void getFooterRows();
```

### Return Value
***array[row object]*** : Array of data row objects

### Example
```javascript
// Returns an array of footer data row objects in the footer area.
sheet.getFooterRows();
```

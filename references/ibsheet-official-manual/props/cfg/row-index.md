# RowIndex ***(cfg)***

> You can change the column name that replaces the `SEQ` functionality. 
 If you specify a column name with this feature, the `SEQ` functionality can be used in that column instead. 



### Type
`string`


### Example
```javascript
options.Cfg = {
    RowIndex: "AAA" // Column name to replace the SEQ functionality
};

```

# TabletMode ***(cfg)***

> 시트를 모바일 터치 모드로 동작하도록 설정합니다.

### Type
`boolean`


### Example
```javascript
options.Cfg = {
    // 모바일 터치 모드로 동작
    TabletMode: 1
};
```

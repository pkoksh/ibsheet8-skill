# UseCache ***(cfg)***

> Sets whether to use browser cache for `img` files.


### Type
`boolean`

### Options
|Value|Description|
|-----|-----|
|`0(false)`|Do not use browser cache (`default`)|
|`1(true)`|Use browser cache|

### Example
```javascript
options.Cfg = {
   UseCache: 1 // Use browser cache
};
```

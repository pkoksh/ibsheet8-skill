# TipMaxWidth ***(cfg)***

> Sets the maximum width (`MaxWidth`) of the tip. 


### Type
`number`


### Options

|Value|Description|
|-----|-----|
|`number`|Maximum width of the tip|

### Example
```javascript
options.Cfg = {
    TipMaxWidth: 500 // Set the maximum width of the tip to 500
};
```

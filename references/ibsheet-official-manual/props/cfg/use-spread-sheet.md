# !UseSpreadSheet ***(cfg)***

> 스프레드 시트 모드를 사용합니다


### Type

`boolean`

### Options

|Value|Description|
|-----|-----|
|`0(false)`|스프레드 시트 모드 사용 안함 (`default`)|
|`1(true)`|스프레드 시트 모드 사용|


### Example

```javascript
options.Cfg = {
  UseSpreadSheet: true      // 스프레드 시트 모드 사용
};
```

# PivotFormat ***(cfg)***

> 피벗 시트에 있는 셀들에 적용될 포맷을 설정합니다.



### Type
`string`

### Example
```javascript
options.Cfg = {
    UsePivot: true, // 피벗 사용 여부
    PivotFormat: "#,### 원" // 시트 내 셀들의 포맷
};
```

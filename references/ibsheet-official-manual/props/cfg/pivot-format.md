# PivotFormat ***(cfg)***

> Sets the format to be applied to cells in the pivot sheet.



### Type
`string`

### Example
```javascript
options.Cfg = {
    UsePivot: true, // Whether to use pivot
    PivotFormat: "#,### Won" // Format of cells in the sheet
};
```

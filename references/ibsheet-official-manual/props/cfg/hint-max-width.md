# HintMaxWidth ***(cfg)***

> 힌트의 최대 너비(`MaxWidth`)을 설정합니다. 


### Type
`number`


### Options

|Value|Description|
|-----|-----|
|`number`|힌트의 최대 너비|

### Example
```javascript
options.Cfg = {
    HintMaxWidth: 500 // 힌트의 최대 너비를 500으로 설정
};
```

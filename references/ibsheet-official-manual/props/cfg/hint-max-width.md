# HintMaxWidth ***(cfg)***

> Sets the maximum width (`MaxWidth`) of hints. 


### Type
`number`


### Options

|Value|Description|
|-----|-----|
|`number`|Maximum width of hints|

### Example
```javascript
options.Cfg = {
    HintMaxWidth: 500 // Set the maximum width of hints to 500
};
```

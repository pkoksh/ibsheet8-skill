# MenuMaxHeight ***(cfg)***

> `(Col, Cell)Type`이 `Enum`인 열에서 열리는 메뉴의 최대 높이를 지정합니다.

> `Suggest`로 열리는 메뉴에도 적용됩니다.

### Type
`string`


### Example
```javascript
options.Cfg = {
    MenuMaxHeight: 400
};
```

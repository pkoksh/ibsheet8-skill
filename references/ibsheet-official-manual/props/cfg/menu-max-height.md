# MenuMaxHeight ***(cfg)***

> Specifies the maximum height of the menu that opens in columns where `(Col, Cell)Type` is `Enum`.

> Also applies to menus opened via `Suggest`.

### Type
`string`


### Example
```javascript
options.Cfg = {
    MenuMaxHeight: 400
};
```

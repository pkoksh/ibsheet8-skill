# version ***(method)***
> Checks the product version of the sheet.

### Syntax
```javascript
string version();
```

### Return Value
***string***: Version of the `ibsheet.js` product loaded on the page and version strings of loaded plugins

### Example
```javascript
// Check the product version of the sheet
IBSheet.version();
```

# version ***(method)***
> 시트의 제품 버전을 확인합니다.

### Syntax
```javascript
string version();
```

### Return Value
***string***: 화면에 로드된 `ibsheet.js` 제품의 버전 및 로드된 플러그인들의 버전 문자열

### Example
```javascript
// 시트의 제품 버전 확인
IBSheet.version();
```

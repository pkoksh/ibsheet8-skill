# onKeyDown ***(event)***
> Event called when the user presses a key in the focused sheet.

> Called for all function keys, and returning `1(true)` ignores the default action of the pressed key (e.g., adding a row via Insert key, switching to edit mode via Enter key, etc.).

> In edit mode, the event fires but the default action cannot be prevented through return.

### Syntax

```
    onKeyDown : function(paramObject) {

    }
or
    sheet.bind("onKeyDown" , function(paramObject) {});
```

### Parameters


| Name | Type | Description |
|----------|-----|-------|
|sheet|`object`|Focused sheet object|
|key|`number`|`keyCode` of the pressed key|
|event|`object`|JavaScript key event generated by `onKeyDown`|
|name|`string`|Name of the pressed key (`A, Esc, Backspace, Enter`, etc.)|
|prefix|`string`|Key name when pressed together with `Shift, Alt, Ctrl` keys
Combined in order of `Shift-> Ctrl-> Alt`.
For example, when `Shift` and `Ctrl` are pressed together, `"ShiftCtrl"` is returned; when `Alt` and `Ctrl` are pressed together, `"CtrlAlt"` is returned|

### Return
***boolean***

### Example
```javascript
options.Events = {
    onKeyDown:function(evtParam){
        // Block Insert key and Delete key input on the focused sheet.
        if(evtParam.name == "Ins" || evtParam.name == "Del") return true;
    }
}
```

# onKeyUp ***(event)***
> Event called when a pressed key is released on the focused sheet.

> Called for all function keys, and fires after the [onKeyPress](./on-key-press) event.

### Syntax

```
    onKeyUp : function(paramObject) {

    }
or
    sheet.bind("onKeyUp" , function(paramObject) {});
```

### Parameters


| Name | Type | Description |
|----------|-----|-------|
|sheet|`object`|Focused sheet object|
|key|`number`|`Unicode` of the pressed key|
|event|`object`|JavaScript key event generated by `onKeyUp`|
|name|`string`|Pressed key name (`A, Esc, Right..`)|
|prefix|`string`|First key name when multiple keys are pressed (`ctrl, shift..`)|

### Return
***none***


### Example
```javascript
options.Events = {
    onKeyUp:function(evtParam){
        alert(evtParam.name + " key was pressed.");
    }
}
```

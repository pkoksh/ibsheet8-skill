# onKeyPress ***(event)***
> Event called when the user presses a key in the focused sheet.

> This event fires after the [onKeyDown](./on-key-down) event, and is only called when pressing character or number keys.

> For character keys, the `unicode` and `name` parameter values are **different** from those of [onKeyDown](./on-key-down), [onKeyUp](./on-key-up).

<!--!
> `[Private note]` Returning true ignores the default action of the pressed key. Since press doesn't have many default actions, keeping this private...
!-->

### Syntax

```
    onKeyPress : function(paramObject) {

    }
or
    sheet.bind("onKeyPress" , function(paramObject) {});
```

### Parameters


| Name | Type | Description |
|----------|-----|-------|
|sheet|`object`|Focused sheet object|
|key|`number`|`Unicode` of the pressed key|
|event|`object`|JavaScript key event generated by `onKeyPress`|
|name|`string`|Pressed key name (`A, Esc, Right..`)|
|prefix|`string`|First key name when multiple keys are pressed (`ctrl, shift..`)|

<!--!
### Return
`[Private]` ***boolean***
!-->

### Example
```javascript
options.Events = {
    onKeyPress:function(evtParam){
        alert(evtParam.name + " key was pressed.");
    }
}
```
